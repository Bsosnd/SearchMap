# SearchPlace

진행상황
- 장소검색 / 검색한 장소의 위치값 받아오기
- 현재위치 받기
- 현재위치로부터 검색된 장소(목적지)까지의 경로 검색하기
- 목적지 주변의 랜드마크들 카테고리를 이용해서 보여주기
